# Python Web开发学习指南

## 学习路径

### 1. Flask框架基础 (01_flask_basics)
- Flask应用结构
- 路由和视图
- 模板渲染
- 表单处理
- 用户认证

### 2. 数据库操作 (02_database)
- SQLAlchemy ORM
- 数据模型设计
- 数据库迁移
- 查询和关系

### 3. RESTful API开发 (03_restful_api)
- API设计原则
- 请求处理
- 响应格式化
- 错误处理
- API认证和授权

### 4. 前后端交互 (04_frontend)
- 静态文件处理
- AJAX请求
- JSON数据交换
- 跨域处理

## 项目实战
- 构建个人博客系统
- 开发RESTful API服务
- 实现用户认证系统

## 学习建议
1. 确保已安装所有必要依赖（见根目录requirements.txt）
2. 按照模块顺序循序渐进学习
3. 每个示例代码都要动手实践
4. 注意数据库安全性和性能优化
5. 多参考官方文档和最佳实践
